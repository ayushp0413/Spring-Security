# YAML / YML file :- Different ways to set properties
Priority of property sources in Spring Boot (from highest to lowest):
1️⃣  Command-line arguments	--server.port=9090 (via CLI or IntelliJ Program Arguments)
2️⃣	Java System properties	-Dserver.port=9090
3️⃣	Environment variables	SERVER_PORT=9090
4️⃣	application.properties / application.yml	Inside src/main/resources
5️⃣	Default values in code	SpringApplication.setDefaultProperties(...)
